# Project Setup

## 1. Create a Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate
```

## 2. Install Dependencies

```bash
pip3 install .
```

## 3. Setup .env
Create a .env file in the root dir, and put your API key there.

## 4. Run an agent

```bash
python3 agent/basic_ops.py
```
